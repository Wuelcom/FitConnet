# FitConnet
Proyecto FitConnet

Sistema web de seguimiento de rutinas, dietas y progreso físico.

## Estructura del proyecto

- `/backend`: API REST con FastAPI
- `/frontend`: Interfaz web
- `/db`: Base de datos MySQL y scripts
- `/docs`: Documentación del proyecto
